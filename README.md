# Manipulating Data in Angular with Kendo UI Data Query

## Project Summary
This project demonstrates advanced data manipulation techniques (sorting, filtering, grouping) using the Kendo UI Data Query library.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. Test the data manipulation functions applied to the product list.
